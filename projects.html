Discovery Project · Team-Based · Spring 2026

Bacter.ai — Clinical Decision Support Tool

Full-stack genomic AI platform for antibiotic resistance prediction
PythonFlaskXGBoostReactViteREST APISHAP

Bacter.ai predicts antibiotic resistance across 10 antibiotics in under 60 seconds from raw FASTA input, replacing a 3-day laboratory process. The project sits at the intersection of machine learning, bioinformatics, and clinical decision support with direct implications for patient outcomes.

I contributed both to the backend ML pipeline and the entire frontend interface, giving me end-to-end ownership of a real software product.

What We Built
  • 10 XGBoost classifiers trained on 4,096-dimension k-mer vectors with 5-fold cross-validation.
  • SHAP explainability pipeline producing antibiograms, ROC curves, and confusion matrices for clinical interpretation.
  • Flask REST API backend handling FASTA ingestion, feature extraction, and real-time prediction.
  • React and Vite frontend with animated resistance mechanism viewer, circular genome map, and live verification panel.
My Contribution
  • Co-developed the complete platform architecture end-to-end across backend and frontend.
  • Built all interactive UI components and the live panel consuming Flask API endpoints in real time.
  • Integrated SHAP outputs into interpretable visualizations within the clinical dashboard.
  • Participated in model validation and the cross-validation framework for all 10 antibiotic classifiers.
